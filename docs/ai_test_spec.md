# src_ai テスト仕様書

## 1. 文書情報
- 対象: src_ai フォルダ配下のCソース
- 作成日: 2026年5月20日
- 参照設計書:
  - [docs/ai_design.md](docs/ai_design.md)
  - [docs/ai_detailed_design.md](docs/ai_detailed_design.md)
- 管理方針: 単体テストと結合テストを本ファイルで一元管理する

## 2. テスト実施ルール
- 各テスト実施後に、実際結果を実際結果欄へ記入する。
- 判定欄には、合格なら◯、不合格なら✕を記入する。
- 不合格時は、備考に再現手順や原因、修正チケット番号を記入する。

## 3. 前提条件
- ビルドコマンド: gcc -finput-charset=utf-8 -fexec-charset=cp932 src_ai/*.c -o app_ai.exe
- 実行ファイル: app_ai.exe
- 作業ディレクトリ: プロジェクトルート
- CSVファイル:
  - src_ai/sales_ai.csv
  - src_ai/operation_log_ai.csv
