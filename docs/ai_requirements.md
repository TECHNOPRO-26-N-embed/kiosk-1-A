# 要件定義書

## ヒアリングメモ
	- 商品の最大在庫数は50個程度
	- 売上記録・操作ログはCSVで保存
	- 取扱商品は飲料限定・最大50種類
	- 季節ごとに商品の入れ替えあり
	- セキュリティは可能な範囲で考慮、システム停止しないこと重視
	- 複数のお客さんが続けて利用できる

## 自動販売機システム 機能要件

| 機能ID | 機能名 | 概要 |
| :---: | :--- | :--- |
| F01 | 商品管理 | 最大50種類の飲料商品を管理し、各商品の在庫数は最大50個まで。季節ごとに商品の入れ替えが可能。 |
| F02 | 売上・操作記録 | 売上記録および操作ログをCSVファイルで保存。記録には日時や商品名、金額、操作内容などを含む。 |
| F03 | 販売機能 | 複数のお客さんが連続して利用できる。商品選択、購入、つり銭返却などの基本操作を提供。 |
| F04 | セキュリティ・安定性 | システムの安定稼働を重視し、エラー処理や不正操作時のログ記録、可能な範囲でのセキュリティ対策を実施。 |
